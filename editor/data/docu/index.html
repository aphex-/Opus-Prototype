Opus prototype

procedural world generation

Opus prototype is a tool to generate infinite landscapes by combining noise artifacts. You can use the editor to create a world and use the library to import it to your own application. Opus prototype offers you a way to easily describe landscapes (like seen in Minecraft) with values that are based on pseudo random noise algorithms.
